* **功能与限制：** 其中 a15 及以上机型越狱后可以解决发热降低亮度并显著缓解温控，但不能像以前的芯片一样完全去除性能限制，手机的性能还会受制于无视温度的功耗锁，就算把手机芯片冷却至 0 度，功耗锁也依旧生效，满功耗状态仅能维持几秒，随后无视温度将功耗锁死在 6w 左右。
* **iOS 26.1 (?) 及以下：** ios17.0.1 以上虽不支持巨魔/越狱，但 ios26.1 及以下可实现缓解发热降低亮度、缓解温控问题（不能彻底去除）、开启国行 AI、开启 iPad 功能、关闭烦人的灵动岛等以前需要越狱才能实现的功能。
* **建议：** 如果你希望自己的苹果能进行有限的玩机，请不要升级系统，就算你是 ios18 系统/17 系统也不要升级 ios18/ios17 的后续小版本，也会被封堵的。

## 华为 (含分家之前的荣耀)

### 麒麟芯片机型
* **麒麟 980 及以前：**
    * **980 以前：** 部分老机型可以找某贼解锁（详见酷安某贼）。
    * **麒麟 980：** **可解**。但升级过鸿蒙的 980 由于熔断，需要买菊花三代（可能需要其他付费软件）才能操作。
* **麒麟 990：** **无解！！**

### 高通芯片机型
* **高通骁龙 865 (MatePad 11)：** **可解**。该平板暂时未推送熔断版本。
* **高通骁龙 8+：**
    * **解锁条件：** 如果是鸿蒙 4，未升级过 `4.0.0.126`；如果是鸿蒙 3，未升级过 `3.1.0.180` 的骁龙 8+ 可解。
    * **熔断机制：** **一旦升级过就永久不可逆熔断且降级无效，地球爆炸了也绝对不可能解锁。**
    * **极度稀缺警告：** 现在市面上绝大多数的华为骁龙 8+ 均已被熔断，你几乎不可能买到未熔断的。
    * **回锁风险：** 华为全部机型一旦更新或降级系统就会被回锁，系统修复也会回锁。除非是在熔断前就 root 了而且一直没升级的，才有可能在今天还保持解锁状态。**由于几乎全部被熔断，该类机型也已经达到了无解标准。**

### 联发科 (MTK) 及其他
* **联发科：** 部分有 mtk 漏洞的联发科可解（酷安 str123 的工具）；但安卓 9 及以下的联发科机型，不一定能用 str123 开发的工具解锁。
* **其余：** 无解。
